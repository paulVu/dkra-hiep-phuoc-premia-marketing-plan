1. Kiến trúc hệ thống

5-Layer OpenClaw Pipeline

V4 thêm Layer E: Reporting tự động. Kasa là não trung tâm điều phối cả 5 tầng.

📥 Layer A
Ingestion web_fetch + pdf → Knowledge Base
🏭 Layer B
Content Factory Claude + image_generate
📡 Layer C
Distribution message tool + cron
🎯 Layer D
Lead Engine Telegram → gog Sheets
📊 Layer E
Auto-Reporting Kasa → Telegram daily KPI
Layer A

📥 Ingestion — Không thủ công

Cách chạy

  • Kasa dùng pdf tool để parse tài liệu dự án
  • web_fetch để thu thập tin thị trường
  • web_search cho competitive intel hàng tuần
  • Kết quả lưu vào knowledge-base.json trong workspace

Trigger

  • Cron mỗi thứ Hai 7h sáng: refresh market intel
  • P upload tài liệu mới → Kasa tự parse ngay
Layer B

🏭 Content Factory — Built-in Claude

Không cần tool ngoài

  • Claude (chính Kasa) sinh 8 output formats từ 1 brief
  • image_generate tạo ảnh branded với overlay
  • Fast model cho drafts → Quality model cho final
  • Kết quả lưu vào content/queue/YYYY-MM-DD.md

Human review

  • Kasa gửi batch qua Telegram để P approve
  • P reply "✅ approve" hoặc edit trực tiếp
  • Sau approve → content vào publish queue
Layer C

📡 Distribution — message tool

Cách phân phối

  • Broker Zalo group: Kasa gửi Telegram → P forward/repost
  • Facebook: Kasa chuẩn bị copy + ảnh → P post (hoặc browser tool)
  • Cron schedule: OpenClaw cron trigger Kasa đúng giờ
  • Broker kit: Kasa push qua Telegram group tự động

Giới hạn thực tế

  • Facebook API cần Page token (P setup 1 lần) → Kasa dùng API trực tiếp
  • Zalo OA: Kasa soạn, P paste (hoặc Zalo API nếu có key)
Layer D

🎯 Lead Engine — gog + Telegram

Flow thu lead

  • Lead inbox Telegram → Kasa classify ngay
  • Kasa log vào Google Sheets via gog skill
  • Hot lead (score ≥ 10) → Kasa notify P ngay lập tức
  • Auto-reply: Kasa gửi kit phù hợp < 5 phút
// Lead record tự động ghi vào Sheets { "timestamp": "2026-04-03 22:00", "name": "Anh Minh", "channel": "Telegram", "intent": "Investor - hỏi giá + ROI", "score": 13, "status": "HOT → notified P", "kit_sent": "Kit Investor v2" }
Layer E — MỚI trong V4

📊 Auto-Reporting — Kasa tự báo cáo

Daily (7h sáng)

  • Đếm leads hôm qua từ Sheets
  • Tóm tắt hot leads cần follow-up
  • Content đăng hôm nay là gì
  • Gửi qua Telegram DM cho P

Weekly (T6 17h)

  • Total reach / engagement / DMs
  • Top 3 winning hooks tuần này
  • Leads qualified: X / target Y
  • Budget paid ads vs results

Alert (real-time)

  • Hot lead vào (score ≥ 10)
  • Form submit từ landing page
  • Cron job fail
  • Content queue sắp hết

4. Lead Engine

Kasa là CRM — gog là database

Không cần HubSpot. Kasa classify intent ngay khi nhận Telegram, lưu Google Sheets, alert P khi hot.

Capture → Classify

Kasa nhận và phân loại ngay

Nguồn lead

  • Telegram DM vào Kasa bot
  • Comment trigger "TÀI LIỆU", "GIÁ", "PHÁP LÝ"
  • Landing page form submit (webhook → Telegram)
  • Zalo OA: Kasa soạn reply, P paste

Phân loại (Claude classify)

  • 🏠 Buyer: ở thực, tiện ích, trường → Kit Buyer
  • 💰 Investor: ROI, giá, khu vực → Kit Investor
  • 🤝 Broker: hoa hồng, hợp tác → Kit Broker
  • ❓ Unclear: Kasa hỏi thêm 1 câu qualifying
gog Sheets CRM

Auto-update

  • Kasa ghi Sheets khi có lead mới
  • Kasa update status khi follow-up xong

Lead Scoring — Kasa tính tự động

Tín hiệu yếu (+1-3)

  • +1 Xem video >50%
  • +2 Save / share bài
  • +3 Comment hỏi thông tin

Tín hiệu mạnh (+5-8)

  • +5 Inbox hỏi giá / chính sách
  • +6 Yêu cầu tài liệu pháp lý
  • +8 Hỏi đặt lịch xem dự án

Alert ngưỡng

  • ≥ 10: 🔥 HOT → Kasa ping P ngay
  • 5-9: Warm → vào nurture queue
  • 1-4: Cold → auto content drip
5. Broker Copilot

Kasa push kit mỗi sáng — broker chỉ paste

Cron 7h sáng, Kasa gửi daily kit vào Telegram group broker. Không cần Zalo manual, không cần Drive share.

Daily Kit — 7h sáng

Broker nhận mỗi ngày

  • 1 bài đăng FB — ready, copy-paste
  • 1 ảnh branded — image_generate render
  • 1 đoạn Zalo — gửi cho khách
  • 3 objection scripts — "Giá cao" → "So sánh..."
  • 1 market tip — 2-3 câu tin tức khu vực

Cách Kasa deliver

  • Cron 7h → Kasa generate kit từ content queue
  • message tool → push vào Telegram group broker
  • Ảnh đính kèm inline, text đã format sẵn
Broker Lead Capture

Lead từ broker → vào CRM

  • Broker nhận lead → forward vào Telegram Kasa
  • Kasa classify + score ngay
  • Log vào Sheets tab Broker Activity
  • Nếu hot: Kasa alert P, gắn broker name để tính hoa hồng

Leaderboard (weekly)

  • Kasa pull Sheets → tổng hợp top brokers tuần
  • Post leaderboard vào group mỗi thứ Sáu
  • Gamification: tạo động lực cạnh tranh tích cực

9. SOP 30 ngày

Triển khai OpenClaw-native từ ngày 1

W0

Setup (1-2 ngày)

Kasa làm:

  • Cài gog skill, kết nối Google Sheets CRM
  • Cài blogwatcher skill, add RSS feeds khu vực Nhà Bè / Long An
  • Parse toàn bộ PDF tài liệu dự án → knowledge-base.json
  • Tạo content templates cho 8 formats
  • Setup cron jobs (daily + weekly)
  • Tạo 3 auto-reply kits: Buyer / Investor / Broker
  • Setup Telegram broker group (P tạo, Kasa làm admin bot)

P làm: Kết nối Google account cho gog skill (5 phút)

W1

Launch — Nhận diện

Goal: 10,000 reach, pipeline chạy trơn tru

  • Kasa generate batch tuần 1 → P approve qua Telegram
  • Cron broker kit bắt đầu chạy lúc 7h sáng
  • Lead intake: Kasa classify mọi DM, ghi Sheets
  • Test 5 hooks → track trong Sheets Content Performance tab
  • Cuối tuần: Kasa gửi KPI report đầu tiên
W2

Thuyết phục & Test

Goal: 50+ saves, 20+ DMs qualified, winners identified

  • Content: thị trường, pháp lý, chính sách
  • A/B test: top 3 hooks × 3 CTAs = 9 variants
  • Bắt đầu paid: boost bài save rate > 3%
  • Kasa update winner playbook từ kết quả thực
  • Broker kit #2: thêm objection scripts từ lead thực
  • Kasa gửi broker leaderboard tuần đầu
W3

Scale & Nurture

Goal: 25+ qualified leads, retarget pipeline active

  • Scale winning formats (Kasa auto-generate nhiều hơn từ best hooks)
  • Nurture: Kasa follow-up DMs chưa reply sau 3 ngày
  • Kasa build lookalike brief cho P setup paid
  • Broker kit #3: case study từ lead thật
  • Kasa report ROI giữa kỳ: chi phí vs kết quả
W4

Convert & Optimize

Goal: 15+ sales handoffs, winner playbook hoàn chỉnh

  • Heavy CTA content: urgency, scarcity, event invite
  • Kasa push toàn bộ warm leads → sales handoff
  • Compile final playbook: hooks, CTAs, timing, formats
  • Monthly ROI report: đầy đủ pipeline từ A đến E
  • Decision: Continue same / double down / expand scope
